# Harness Adapters

This directory contains per-harness adapter metadata and command mappings generated from the core Rubber Duck command specification.

Each harness folder includes:

- `adapter.yaml` — static metadata + command map
- `README.md` — harness-specific install/mapping notes

Harnesses in scope:

- `claude-code`
- `copilot-cli`
- `codex`
- `cursor`
- `windsurf`
- `gemini-antigravity`
